## ✅ שלב 2: בדיקת אתר בדפדפן

   - [ ] האתר נטען ללא שגיאות

   - [ ] התפריט עובד

## ✅ שלב 3: בדיקת טופס יצירת קשר

1. **גלול לטופס "יצירת קשר"**
2. **נסה לשלוח טופס ריק** → לחץ "שלח הודעה"
   - [ ] **צפוי:** הודעת שגיאה מופיעה
   
3. **מלא רק שם (פחות מ-2 תווים)** → נסה לשלוח
   - [ ] **צפוי:** "שם מלא חובה (מינ׳ 2 תווים)"
   
4. **מלא שם אבל ללא email/phone** → נסה לשלוח
   - [ ] **צפוי:** "חובה למלא אימייל או טלפון אחד לפחות"
   
5. **מלא אימייל לא תקין** → נסה לשלוח
   - [ ] **צפוי:** "אימייל לא תקין"
   
6. **מלא הכל נכון:**
   - שם: "דני כהן" (מעל 2 תווים)
   - אימייל: "test@example.com"
   - קטגוריה: בחר "נרות"
   - כמות: 2
   - הודעה: "אני מעוניין בנר מותאם אישית" (מעל 3 תווים)
   
   לחץ "שלח הודעה"
   - [ ] **צפוי:** הודעת הצלחה "ההודעה נשלחה..."
   - [ ] **צפוי:** הטופס התאפס (ריק)

---

## ✅ שלב 4: בדיקת חיפוש מוצרים

1. **לחץ על אייקון החיפוש** (בתפריט העליון)
2. **הקלד "ALLISON"** → לחץ Enter
   - [ ] **צפוי:** מוצר "ALLISON SET" מופיע בתוצאות
   
3. **לחץ על המוצר**
   - [ ] **צפוי:** האתר גולל לקטע הגלריה
   
4. **נסה להוסיף לסל** מתוצאות החיפוש
   - [ ] **צפוי:** המוצר מתווסף לעגלה

---

## ✅ שלב 5: בדיקת עגלת קניות

1. **לחץ על אייקון העגלה**
2. **בדוק:**
   - [ ] אם יש מוצרים - הם מופיעים
   - [ ] אם אין - "Your cart is empty"
   
3. **אם יש מוצרים:**
   - שנה כמות (עם +/-)
   - [ ] **צפוי:** מחיר מתעדכן
   - לחץ "המשך לתשלום"
   - [ ] **צפוי:** דף תשלום נפתח

## ✅ שלב 6: בדיקת דף תשלום

1. **מצב משלוח (שלב 1):**
   - מלא שם מלא
   - מלא טלפון
   - מלא כתובת
   - מלא עיר
   - לחץ "המשך לתשלום"
   - [ ] **צפוי:** עובר לשלב 2
   
2. **מצב תשלום (שלב 2):**
   - בחר "כרטיס אשראי"
   - [ ] **צפוי:** שדות כרטיס מופיעים
   - בחר "תשלום בעת המשלוח"
   - [ ] **צפוי:** הודעה על תשלום במשלוח
   - לחץ "המשך לסיכום"
   - [ ] **צפוי:** עובר לשלב 3
   
3. **סיכום (שלב 3):**
   - [ ] **צפוי:** רשימת מוצרים מופיעה
   - [ ] **צפוי:** פרטי משלוח מופיעים
   - [ ] **צפוי:** מחיר כולל מופיע
   - לחץ "אשר והזמן"
   - [ ] **צפוי:** הודעת הצלחה עם מספר הזמנה

## ✅ שלב 7: בדיקת הרשמה

1. **לחץ על אייקון המשתמש**
2. **בחר "הרשם כאן"**
3. **נסה להרשם עם:**
   - שם קצר (1 תו) → **צפוי:** שגיאה
   - אימייל לא תקין → **צפוי:** שגיאה
   - סיסמה קצרה מ-6 תווים → **צפוי:** שגיאה
   - סיסמאות לא תואמות → **צפוי:** שגיאה

4. **מלא הכל נכון:**
   - שם מלא
   - אימייל תקין
   - סיסמה (6+ תווים)
   - אישור סיסמה (תואם)
   - לחץ "הירשם"
   - [ ] **צפוי:** הודעת הצלחה

## ✅ שלב 8: בדיקת אבטחה (DevTools)

1. **פתח DevTools** (F12)
2. **לך ל-Network tab**
3. **רענן את הדף**
4. **בחר בקשה כלשהי → Headers**
5. **בדוק Response Headers:**
   - [ ] `X-Content-Type-Options: nosniff`
   - [ ] `X-Frame-Options` קיים
   - [ ] `Content-Security-Policy` קיים
