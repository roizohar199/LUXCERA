# 🎛️ Admin Panel - LUXCERA CMS

ממשק ניהול לבעל האתר לניהול מוצרים, תמונות ומחירים.

## 📍 גישה ל-Admin Panel

**כתובת:** `http://localhost:5173/admin.html`

בפרודקשן: `https://your-domain.com/admin.html`

## 🔐 פרטי התחברות

**ברירת מחדל:**
- שם משתמש: `admin`
- סיסמה: `admin123`

⚠️ **חשוב מאוד:** שנה את הסיסמה בפרודקשן!

## 🚀 שימוש

1. **התחברות:**
   - פתח את `http://localhost:5173/admin.html`
   - הזן שם משתמש וסיסמה
   - לחץ "התחבר"

2. **הוספת מוצר חדש:**
   - לחץ על "הוסף מוצר חדש"
   - מלא את הפרטים:
     - שם המוצר (חובה)
     - קטגוריה (מארזים/נרות אח/פניני שעווה/אביזרים)
     - תיאור (אופציונלי)
     - מחיר (חובה)
     - מחיר מבצע (אופציונלי)
     - תמונה (חובה) - ניתן להעלות קובץ או להזין URL
   - סמן "מוצר פעיל" אם ברצונך שהוא יופיע באתר
   - לחץ "שמור"

3. **עריכת מוצר:**
   - לחץ על "ערוך" ליד המוצר
   - שנה את הפרטים הרצויים
   - לחץ "עדכן"

4. **מחיקת מוצר:**
   - לחץ על כפתור המחיקה (🗑️) ליד המוצר
   - אשר את המחיקה

5. **העלאת תמונה:**
   - בטופס המוצר, לחץ על "העלה קובץ"
   - בחר קובץ תמונה (JPG, PNG, GIF, WebP)
   - התמונה תועלה אוטומטית

## 🔧 אפשרויות נוספות

- **הסתרת מוצר:** בטל את הסימון "מוצר פעיל" - המוצר יישאר במערכת אבל לא יופיע באתר
- **מבצעים:** הזן "מחיר מבצע" נמוך מהמחיר הרגיל - המוצר יציג את המחיר במבצע

## 🔒 אבטחה

- כל הפעולות דורשות התחברות
- הטוקן נשמר ב-localStorage ונשלח בכל בקשה
- בפרודקשן, שנה את `JWT_SECRET` בקובץ `.env` של השרת

## 📝 שינוי סיסמה (מתקדם)

לשינוי סיסמת המנהל, צריך לעדכן ישירות ב-database:

1. פתח את `server/data/luxcera.db` עם כלי SQLite
2. עדכן את הטבלה `admin_users`

או צור סקריפט Node.js שיעשה זאת.
